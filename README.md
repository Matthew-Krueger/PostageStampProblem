# Postage Stamp  Problem
Attempts to explain the Postage Stamp Problem also called the Chicken McNuggets Theorem

## Building
To build this project, you must have cmake and a C++17 compiler. Under Scripts is a script that will build and fun the program automatically. To run it, run it from the root directory of the project. I.E. be in the root of the project, and run `sh Scripts/Build.sh`
